# 长篇写到 50 万字，设定不崩

写到几十万字，最怕设定自己都记不清、前后打架。思衍从架构上解决这件事：

- **全局时间线** — 所有事件按时间排成一张表，写到哪都不会乱
- **稳定的事件编号** — 第 3 章埋的伏笔，到第 <span data-count="200">200</span> 章还认得出来
- **结局一旦定下就锁死** — 不会跑着跑着烂尾

人物、世界观、主线，前后都对得上。
