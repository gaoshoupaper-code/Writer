# AI 智能写作

Writer 的核心是**多 Agent 协作**的写作引擎。

不是一个简单的"提示词→文本"工具，而是一套完整的创作流水线：

- **访谈 Agent**：通过结构化问答收集你的创作需求
- **大纲 Agent**：生成世界观、主线、人物关系
- **章节 Agent**：按大纲逐章生成正文
- **审查 Agent**：质量把关与风格统一

每个 Agent 各司其职，产出可追溯、可回溯。
