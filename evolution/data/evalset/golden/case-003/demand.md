# 创作需求文档

## 核心层

- **体裁类型**（可主类型 + 次类型融合）：
  - 主类型：玄幻
  - 次类型 / 融合元素：日常种田流 + 轻度美食元素 + 治愈系
- **核心创意与卖点（含金手指/外挂设定）**：
  - 一句话故事（logline）：灵厨世家的废灵根少女继承了一家濒临倒闭的深山客栈，靠一手能把普通食材化腐朽为神奇的灵膳手艺，以及一颗温暖待人的心，让荒僻小客栈成了修仙界各路大人物偷偷下凡"寻一口家乡味"的圣地——这里没有刀光剑影，只有灶台上的烟火气和人心被治愈的温柔。
  - 核心卖点 / 爽点钩子（读者冲着什么来：爽感来源 / 情绪钩子 / 新鲜感）：
    - 爽感：美食描写带来的"通感满足"（色香味跃然纸上，让人想吃饭）；化废为宝的"匠心小确幸"（区别于 case-001 的战斗反差爽，这里是生活满足感）
    - 钩子：每位神秘食客背后都有一段被美食治愈的故事，单元剧式温情（类似深夜食堂的玄幻版）
    - 新鲜感：玄幻世界里的"慢生活"——不升级不争霸，把做饭和待客做成核心，反套路
  - 金手指 / 外挂设定：
    - 来源 / 触发条件：继承祖母遗物——一枚"知味铜钱"，含在舌下能感知食材的最佳搭配与火候
    - 核心能力 / 边界（能做什么、不能做什么）：
      - 能：感知任何食材的最佳搭配、火候、处理方式，激发食材隐藏的风味
      - 能：做出带"灵气滋养"的料理，让食客身心舒缓、记忆回溯温暖瞬间
      - 不能：无法凭空创造食材，不能做毒膳，不能控制他人意志
    - 代价 / 限制（避免无敌，平衡点在哪）：铜钱需以料理人的"心意"驱动——若带着功利心做饭，灵气会反噬（菜变难吃）；做一道大菜需耗费心神，一天最多三道灵膳
    - 成长 / 演化路径（如何升级或转向）：从感知食材 → 感知食客"想吃的味道"（读心式共情）→ 最终领悟"灵膳的真谛是心意而非技法"
- **篇幅档位**（≤20 / 21-50 / 51-90 / 91-130 / 131-170 / 171-200 章）：21-50章
- **目标配比**（主线 / 支线 / 角色线 / 暗线）：主线2 / 支线5 / 角色线2 / 暗线0

## 设定层

- **世界观倾向**：
  - 时空 / 时代背景：架空修仙界"云岚大陆"，但视角远离宗门争霸，聚焦一处名为"忘忧谷"的深山避世之地
  - 核心设定（力量体系 / 社会结构 / 科技水平 / 规则法则，按体裁取舍）：
    - 修炼体系：练气→筑基→金丹→元婴（四境为主，本作不涉及太高的战力层次，修仙是背景不是主线）
    - 灵膳体系：灵材入膳能滋养修为、疗愈内伤、舒缓心魔；灵厨是修仙界小众但受人尊敬的职业
    - 社会结构：外界宗门争斗不休，但忘忧谷因地理隔绝与"灵兽庇护"自成世外桃源
    - 规则法则：食材有灵性，需以诚待之；食物是连接人与人、人与灵兽情感的媒介
  - 独特质感 / 差异化卖点（这个世界与同类作品最大的不同）：这是一个"修炼只是背景板"的玄幻世界，核心是人情与美食；没有反派大 Boss，冲突来自"人心的隔阂"和"生活的琐碎"
- **主角设定倾向**：
  - 身份与起点处境：温菱，16 岁，灵厨世家"温氏"后人，灵根是罕见的"废品食灵根"（只对食材有感应，无法正常修炼），父母外出寻食材失踪多年，独自继承深山"忘忧客栈"
  - 核心欲望（最想要什么）：守住祖母留下的客栈，做出让每个走进客栈的人都感到被善待的饭菜（朴素的人情温暖）
  - 核心恐惧 / 弱点 / 软肋：怕孤独（父母失踪的阴影）；怕自己做的菜留不住人（自我怀疑）
  - 人物弧光方向（起点 → 终点：从什么样的人变成什么样的人）：从"守着回忆独自硬撑的孤独少女"→"用美食和真心凝聚起一个温暖小家的灵膳师"
  - 初始能力 / 资源定位（开局强弱、起跑线在哪）：练气一层（几乎无战斗力），但厨艺天赋极高、心思细腻
- **配角与关系网倾向**：
  - 关键配角原型（盟友 / 对手 / 导师 / 情感线 / 反派等功能位）：
    - 盟友/伙伴：客栈伙计"阿黄"——实为封印记忆的上古灵兽"黄耳犬"，化形为憨厚少年帮忙打杂，温菱的忠实陪伴
    - 导师：祖母留下的《温氏灵膳手札》残卷，每解锁一道菜需完成一个"食材奇遇"
    - 食客群像（单元剧核心）：渡劫失败隐居的老剑仙（被一碗汤唤醒归乡心）、离家出走的宗门小公主（想家）、误入谷底的魔修（本心未泯）等
    - 情感线：暂无固定恋爱线，重心在"众人的温情羁绊"
  - 核心关系张力（哪些关系是冲突或羁绊的来源）：温菱与食客的"一饭之缘与长久羁绊"；温菱与失踪父母的"等待与寻找"（轻悬念线）

## 风格层

- **基调与情绪**：
  - 情绪基调词（如燃 / 压抑 / 轻松 / 治愈 / 悬疑 / 黑色幽默）：治愈 + 温暖 + 烟火气，偶有淡淡的忧伤与离别
  - 整体氛围：温馨明亮，如山间清晨的暖阳；冲突克制，重在"以食化怨"的温柔力量
- **【爽感替代说明】**（本 case 与常规爽文的爽感来源差异，请创作与评估时注意）：
  - 本作**不以"打脸/逆袭/升级"为爽点来源**，而是以以下三种"满足感"替代：
    1. **收获满足**：化废为宝、食材奇遇、一道菜的诞生的成就感（每章一个小满足）
    2. **温馨治愈**：食客被美食治愈记忆、人心隔阂化解的感动（每2-3章一个温情高潮）
    3. **慢节奏解压**：烟火日常、山间生活、灵兽互动的治愈感（贯穿始终）
  - **创作要求**：节奏可慢，但要有"小确幸密度"（每章至少一个温暖的细节/小满足），避免流水账式的平淡。
  - **评估提示**：rubric「爽点密度」维度应以本作的"小确幸密度"为对应参照，而非硬套热血爽文的"每章1-2打脸/逆袭"基线；「套路契合」应考察种田/治愈流套路（美食化人、单元温情），而非热血升级套路。
- **目标受众**：
  - 读者画像：18-35，女性偏多但也欢迎轻松向男性读者，喜欢治愈/日常/美食/种田流，压力大寻求慰藉
  - 阅读场景（通勤碎片 / 深夜沉浸 / 爽感解压等）：睡前治愈、疲惫时的精神慰藉，每章是一个温暖的"小栖息"
- **叙事视角与节奏**：
  - 视角（第一人称 / 第三人称限制 / 全知）：第三人称限制（聚焦温菱，偶尔切食客视角讲他们背后的故事）
  - 节奏（快节奏爽文 / 慢热沉浸 / 张弛交替）：慢热沉浸，单元剧结构（每几章一个食客故事），整体推进靠"客栈名气渐起"和"父母线索"
  - 信息密度 / 留白倾向：信息密度低，重氛围与情感，留白多（给人想象与回味的空间）
- **对标 / 参考作品**（可多部）：
  - 作品与借鉴点（学它什么）：借鉴《深夜食堂》的"以食为媒、单元剧治愈"结构内核；借鉴种田流网文的"慢节奏经营"质感
  - 规避点（不想写成什么样）：不要强行塞入热血升级剧情；不要主角突然获得强战力；不要把治愈写成矫情（克制、真实的温暖）

## 约束层

- **主题与价值观**：
  - 中心主题（故事想探讨什么）：真正的强大不是力量，而是温暖人心的能力；平凡日子里的善意，比刀剑更抚慰人心
  - 价值观立场（主角 / 故事站在哪一边）：肯定"善良""匠心""人与人之间的真诚"，否定"唯力量论""功利至上"
- **禁忌与红线**：
  - 内容禁区：不写血腥暴力；不写黑暗向（背叛/虐心）；不写敏感的"食物下毒控制"情节
  - 平台 / 审查敏感项：美食描写真实诱人但不过度（防暴食敏感）；避开政治隐喻
