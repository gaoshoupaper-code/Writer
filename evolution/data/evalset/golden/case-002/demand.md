# 创作需求文档

## 核心层

- **体裁类型**（可主类型 + 次类型融合）：
  - 主类型：玄幻
  - 次类型 / 融合元素：凡人流 + 鉴宝/商战元素
- **核心创意与卖点（含金手指/外挂设定）**：
  - 一句话故事（logline）：灵根驳杂的散修少年天生异瞳，能看穿一切灵材丹药的瑕疵与真伪，从坊市最底层的灵材鉴定学徒做起，靠一双毒眼和三十年苦熬，在尔虞我诈的修仙界里一步步爬向丹道巅峰——证明修仙界最硬的底牌不是天赋灵根，而是把别人看不上的废材用到极致的眼力与耐心。
  - 核心卖点 / 爽点钩子（读者冲着什么来：爽感来源 / 情绪钩子 / 新鲜感）：
    - 爽感：鉴宝打眼、识破奸商以次充好的"信息差碾压"爽；把废料炼出极品丹的反转感（区别于 case-001 的战斗反差爽，这里是智力/眼力爽）
    - 钩子：每一件有故事的灵材背后都藏着秘密（古修士遗物、上古药园线索），牵出长线谜团
    - 新鲜感：金手指是"信息差型"而非"力量型"，主角靠知识和眼力而非战斗力立足
  - 金手指 / 外挂设定：
    - 来源 / 触发条件：天生异瞳，出生时逢灵兽山脉兽潮，灵气乱流冲刷襁褓，瞳孔变异
    - 核心能力 / 边界（能做什么、不能做什么）：
      - 能：透视灵材/丹药内部结构、瑕疵分布、药性流失程度、年份真伪、最佳处理火候
      - 能：看穿伪造灵材的赝品手段，辨别以次充好的障眼法
      - 不能：无法看穿活物（人/兽）的内心、修为或战力
      - 不能：无法凭空提升自身修为，眼力是辅助不是力量
    - 代价 / 限制（避免无敌，平衡点在哪）：深度透视消耗瞳力，连续使用超半个时辰则暂时失明数日；强行透视高于自身境界太多的仙品会反噬伤眼
    - 成长 / 演化路径（如何升级或转向）：只能看凡品灵材 → 灵药 → 仙草 → 最终看穿天道"命数瑕疵"（无器之眼）
- **篇幅档位**（≤20 / 21-50 / 51-90 / 91-130 / 131-170 / 171-200 章）：21-50章
- **目标配比**（主线 / 支线 / 角色线 / 暗线）：主线4 / 支线2 / 角色线1 / 暗线1

## 设定层

- **世界观倾向**：
  - 时空 / 时代背景：架空修仙界"玄明界"，以底层散修视角切入，重心在坊市交易、黑市捡漏与废矿探险，而非宗门核心
  - 核心设定（力量体系 / 社会结构 / 科技水平 / 规则法则，按体裁取舍）：
    - 修炼体系：练气→筑基→金丹→元婴→化神（五境为主，凡人到化神，大乘以上是传说）
    - 灵材分级：凡材→灵材→仙料→神物（四阶，对应丹药品质上限）
    - 社会结构：大宗门垄断上品灵脉，散修只能在坊市、黑市、废矿里捡漏为生；炼丹师地位尊崇但多被大势力豢养
    - 规则法则：丹道讲究"火候与药性"，灵材的瑕疵处理是炼丹成败关键（呼应金手指的核心价值）
  - 独特质感 / 差异化卖点（这个世界与同类作品最大的不同）：把"灵材鉴定与丹道"做成主角的核心修炼路径，战斗力是短板需靠知识弥补；世界资源博弈围绕"废矿残脉"和"上古丹方残片"展开，而非神器秘境争夺
- **主角设定倾向**：
  - 身份与起点处境：苏沉，15 岁，父母双亡的散修孤儿，灵根驳杂（三系下品），在青岩坊市靠给奸商铺子验货换口饭吃
  - 核心欲望（最想要什么）：活得久、活得稳（凡人最朴素的怕死）；后期升华为"查清父母究竟死于何人之手"
  - 核心恐惧 / 弱点 / 软肋：怕穷（没钱买修炼资源就止步不前）；怕卷入大势力的斗争（蝼蚁之感、牵连身边人）
  - 人物弧光方向（起点 → 终点：从什么样的人变成什么样的人）：从"苟且偷生、只求自保的小人物"→"以眼力和耐心立足、敢于与天争命的丹道宗师"
  - 初始能力 / 资源定位（开局强弱、起跑线在哪）：练气二层（偏低），战斗力弱；但眼力和灵材知识远超同龄人（坊市实战磨出来的）
- **配角与关系网倾向**：
  - 关键配角原型（盟友 / 对手 / 导师 / 情感线 / 反派等功能位）：
    - 导师：坊市老瞎子"盲叟"，表面是算命瞎子，实为当年丹道大宗"玄丹阁"的长老，因一场炼丹事故瞎眼隐退，是主角眼力与丹道的启蒙者
    - 对手/反派：大宗门天才炼丹师"陆景元"，年轻金丹期，傲慢，背后势力打压散修丹师；后期揭露与主角父母之死有关
    - 盟友：采药女散修"阿萝"，同样出身底层，擅长辨识野生灵药，与主角合作跑山采药
    - 情感线：阿萝（底层相濡以沫的慢热温情，不喧宾夺主）
  - 核心关系张力（哪些关系是冲突或羁绊的来源）：苏沉与陆景元的"散修 vs 宗门""眼力 vs 天赋"之争；苏沉与盲叟的"传承与遗憾"

## 风格层

- **基调与情绪**：
  - 情绪基调词（如燃 / 压抑 / 轻松 / 治愈 / 悬疑 / 黑色幽默）：沉 + 务实 + 谨慎，偶有破局的快意（凡人流典型基调）
  - 整体氛围：现实冷峻，修仙界残酷（散修如蝼蚁），但主角靠智慧和耐心凿出一条路，有"苦尽甘来"的踏实感，不是无脑爽
- **目标受众**：
  - 读者画像：18-35 男性为主，喜欢凡人流、看重主角靠智商/布局而非无脑战斗，能接受慢热
  - 阅读场景（通勤碎片 / 深夜沉浸 / 爽感解压等）：沉浸式阅读，喜欢慢热但每几章有"鉴宝识破/布局回收"的小高潮
- **叙事视角与节奏**：
  - 视角（第一人称 / 第三人称限制 / 全知）：第三人称限制（聚焦苏沉，偶尔切对手视角制造信息差）
  - 节奏（快节奏爽文 / 慢热沉浸 / 张弛交替）：张弛交替（凡人流典型节奏，非快爽也非纯慢），关键节点有"打脸/破局"的小爆发，但整体推进靠积累
  - 信息密度 / 留白倾向：信息密度高（灵材知识、丹道细节多），伏笔多且长线回收，留白克制
- **对标 / 参考作品**（可多部）：
  - 作品与借鉴点（学它什么）：借鉴《凡人修仙传》的"凡人谨慎、资源积累、伏笔绵长"骨架；借鉴《修真门派掌门路》的底层经营质感
  - 规避点（不想写成什么样）：不要主角突然获得逆天战力；不要后宫；不要金手指解决一切（眼力是辅助，苦修和布局才是根本）

## 约束层

- **主题与价值观**：
  - 中心主题（故事想探讨什么）：在天赋至上的世界里，普通人靠极致的专业、谨慎与坚持，也能走出一条通天之路
  - 价值观立场（主角 / 故事站在哪一边）：肯定"匠心""凡人的尊严""知识即力量"，否定"血统/天赋决定论""强权掠夺"
- **禁忌与红线**：
  - 内容禁区：不写血腥虐杀；不写未成年恋爱线（主角 15 岁但有成长跨度）
  - 平台 / 审查敏感项：避开政治隐喻；黑市交易描写控制在合理尺度，不美化犯罪
