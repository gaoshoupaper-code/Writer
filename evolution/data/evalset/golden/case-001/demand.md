# 创作需求文档

## 核心层

- **体裁类型**（可主类型 + 次类型融合）：
  - 主类型：玄幻
  - 次类型 / 融合元素：热血升级流 + 轻度系统流
- **核心创意与卖点（含金手指/外挂设定）**：
  - 一句话故事（logline）：一个被家族除名的废物少年，意外觉醒「万器图录」——能解析并复刻世间一切兵器技法，从底层铁匠铺一路打上九天神域，证明真正的强者不是天赋，而是把每一件兵器都用到极致的匠心。
  - 核心卖点 / 爽点钩子（读者冲着什么来：爽感来源 / 情绪钩子 / 新鲜感）：
    - 爽感：主角用最普通的兵器打出最炸的效果（铁锤破神剑），以弱胜强的反差爽
    - 钩子：每解锁一件新兵器都有独特的战斗方式 + 背后故事，收集感强
    - 新鲜感：把「锻造/兵器」做核心金手指，区别于常见的功法/血脉/丹药流
  - 金手指 / 外挂设定：
    - 来源 / 触发条件：少年在被除名当晚，暴雨中躲进废弃祖祠，鲜血滴在祖传残破铁锤上觉醒
    - 核心能力 / 边界（能做什么、不能做什么）：
      - 能：触摸任何兵器 → 「万器图录」自动解析其技法精髓 → 短暂复刻使用
      - 能：长期修炼某件兵器 → 彻底掌握，化为本能
      - 不能：无法凭空创造兵器，只能复刻已接触的
      - 不能：复刻强度受自身修为限制（修为不够，复刻的神器只能发挥三成威力）
    - 代价 / 限制（避免无敌，平衡点在哪）：每次复刻消耗神识，过度使用会神识枯竭昏迷；同时间最多复刻三件兵器
    - 成长 / 演化路径（如何升级或转向）：从只能复刻凡器 → 灵器 → 法器 → 神器，最终目标是融合万器创出自己的「无器之道」
- **篇幅档位**（≤20 / 21-50 / 51-90 / 91-130 / 131-170 / 171-200 章）：21-50章
- **目标配比**（主线 / 支线 / 角色线 / 暗线）：主线5 / 支线1 / 角色线1 / 暗线0

## 设定层

- **世界观倾向**：
  - 时空 / 时代背景：架空玄幻大陆「九州」，冷兵器文明，无火器无电力，有灵气修炼体系
  - 核心设定（力量体系 / 社会结构 / 科技水平 / 规则法则，按体裁取舍）：
    - 修炼体系：淬体→凝气→筑基→金丹→元婴→化神→渡劫→大乘（八境，凡人到大能）
    - 兵器分级：凡器→灵器→法器→仙器→神器（五阶，对应修炼境界威力上限）
    - 社会结构：宗门林立，世家把持资源，散修艰难；锻造师地位尊崇但被大势力控制
    - 规则法则：神识是修炼根基，神识强者可御使更多兵器（呼应金手指的代价）
  - 独特质感 / 差异化卖点（这个世界与同类作品最大的不同）：把「兵器锻造与运用」做成核心修炼路径，不是辅助技能；世界的资源争夺围绕「神兵矿脉」和「上古器灵」展开
- **主角设定倾向**：
  - 身份与起点处境：林铮，16 岁，林家旁支子弟，因觉醒测验「无属性灵根」被嫡系除名，贬去铁匠铺当学徒
  - 核心欲望（最想要什么）：向除名他的家族证明自己，夺回母亲遗物（被嫡系霸占的祖传锻造笔记）
  - 核心恐惧 / 弱点 / 软肋：害怕再次被抛弃（童年母亲早逝、家族驱逐的创伤）；神识枯竭时的失控让他恐惧失去自我
  - 人物弧光方向（起点 → 终点：从什么样的人变成什么样的人）：从「执着于被家族认可的自卑少年」→「明白强大无需他人定义，找到自己锻造之道的宗师」
  - 初始能力 / 资源定位（开局强弱、起跑线在哪）：淬体三重（最低），但手感和悟性极佳（铁匠铺学徒的基础），觉醒万器图录后起步
- **配角与关系网倾向**：
  - 关键配角原型（盟友 / 对手 / 导师 / 情感线 / 反派等功能位）：
    - 导师：铁匠铺老掌柜「赵铁山」，表面粗鄙实则是隐退的神级锻造师，第一个看出林铮天赋的人
    - 对手/反派：林家嫡系天才「林玄霜」，冷酷的家族利益维护者，持有母亲遗物
    - 盟友：散修少女「苏小晚」，同样是世家弃子，与林铮相互扶持
    - 情感线：苏小晚（并肩作战的革命情感，慢热，不喧宾夺主）
  - 核心关系张力（哪些关系是冲突或羁绊的来源）：林铮与林玄霜的「血脉 vs 才能」之争；林铮与赵铁山的「师徒传承」；林铮与苏小晚的「同类相惜」

## 风格层

- **基调与情绪**：
  - 情绪基调词（如燃 / 压抑 / 轻松 / 治愈 / 悬疑 / 黑色幽默）：燃 + 底层逆袭的爽快感，偶有温情
  - 整体氛围：热血昂扬，但底层逻辑扎实（修炼和锻造都有规则），不是无脑爽
- **目标受众**：
  - 读者画像：18-35 男性为主，喜欢玄幻升级流，看重金手指的新颖度和战斗的爽感
  - 阅读场景（通勤碎片 / 深夜沉浸 / 爽感解压等）：通勤碎片 + 睡前解压，每章有小爽点，三五章一个大高潮
- **叙事视角与节奏**：
  - 视角（第一人称 / 第三人称限制 / 全知）：第三人称限制（聚焦林铮，偶尔切对手视角制造信息差）
  - 节奏（快节奏爽文 / 慢热沉浸 / 张弛交替）：快节奏爽文，开篇即冲突（除名），不慢热
  - 信息密度 / 留白倾向：信息密度中等，金手指规则逐步揭示不全盘托出，留悬念
- **对标 / 参考作品**（可多部）：
  - 作品与借鉴点（学它什么）：借鉴《斗破苍穹》的「废材逆袭 + 金手指升级」骨架节奏；借鉴《凡人修仙传》的力量体系严谨性
  - 规避点（不想写成什么样）：不要无脑后宫；不要金手指一上来就无敌（要有清晰的代价和成长曲线）；不要后期战力崩坏

## 约束层

- **主题与价值观**：
  - 中心主题（故事想探讨什么）：真正的强大源于对一事物的极致钻研与热爱，而非天赋血统
  - 价值观立场（主角 / 故事站在哪一边）：肯定「匠人精神」「后天努力」，否定「血统论」「强权即真理」
- **禁忌与红线**：
  - 内容禁区：不写未成年恋爱线（主角 16 岁但有成长跨度）；不写血腥虐杀；不写反社会价值观
  - 平台 / 审查敏感项：避开政治隐喻；暴力描写控制在网文尺度内（打斗但不血腥猎奇）
